# Практикум по программированию на Java (Academic Practice)

Сборник лабораторных и практических работ, выполненных в рамках изучения алгоритмического программирования, объектно-ориентированного подхода (ООП) и разработки пользовательских интерфейсов на языке Java.

## 🛠 Общий технологический стек
* **Язык:** Java (Core / JDK)
* **Интерфейсы:** Java Swing, AWT
* **Работа с данными:** File I/O (Потоки ввода-вывода, сериализация, парсинг текстовых данных)
* **Парадигмы:** Структурное программирование, ООП

---

## 📚 Каталог выполненных работ

### 1. Решение систем линейных алгебраических уравнений (СЛАУ)
* **Папка:** `linear-systems-solver`
* **Описание:** Реализация численных методов для решения математических уравнений. Работа показывает понимание математических алгоритмов и умение переносить формулы в код.

### 2. Табулирование функций и файловый ввод/вывод
* **Папка:** `function-tabulation-io`
* **Описание:** Расчет значений математических функций на заданном интервале с шагом. Реализован экспорт полученной таблицы в текстовые файлы и импорт данных для последующей обработки.

### 3. Визуализация данных и табличные интерфейсы
* **Папка:** `data-visualization-gui`
* **Описание:** Разработка графического пользовательского интерфейса (GUI) на Swing. Включает работу с табличными моделями (`TableModel`) и графическое отображение результатов расчетов.

### 4. Комплексная обработка файлов и структур данных
* **Папка:** `advanced-file-processing`
* **Описание:** Углубленная работа с файловой системой, обработка ошибок ввода-вывода (Exception Handling), фильтрация и валидация данных при чтении из файлов.

## ⚙️ Запуск любого из проектов

Каждая папка представляет собой независимый модуль с собственным исходным кодом.

1. Клонируйте общий репозиторий:
   ```bash
   git clone https://github.com/DanilD12/java-academic-practice.git
2. Откройте нужную папку проекта (например, 02-function-tabulation-io) в вашей среде разработки (NetBeans, IntelliJ IDEA, Eclipse).
3. При наличии папки lib внутри проекта — подключите содержащиеся в ней .jar библиотеки к вашей IDE.
4. Найдите класс, содержащий метод public static void main(String[] args), и запустите его.
